3 reasons to get The Currency Switch on your New Tab

Travelling to a foreign country ?

If you have dollars that need to be converted into the local currency of the country you're visiting, you'd need to know the dollar to euro conversion if you're going to parts of Europe or the dollar to yen conversion if you're traveling to Japan! Add The Currency Switch to be able to use the currency converter tool in situations like these.
